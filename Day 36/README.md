# Cognitive Pattern Explorer 🧠

A single-file, offline-capable web app for reflecting on your thinking patterns — built as **Day 36** of the **60 Days Claude AI Challenge**.

## What it does

Cognitive Pattern Explorer is a calm, game-like self-reflection experience (not a test, and not a diagnosis) that explores five everyday thinking tendencies:

- **Analytical Thinker**
- **Emotional Intuitive**
- **Overthinking Loop Style**
- **Action-First Decision Maker**
- **Balanced Reflective Thinker**

Through three interactive chapters, it maps your responses and generates a personalized reflection journal with a percentage breakdown of your patterns.

## Flow

1. **Start Screen** — choose a Calm or Stress atmosphere
2. **Chapter 1** — Discover Your Thinking Style (scenario-based choices)
3. **Chapter 2** — Choose Your Priorities (draggable sorting cards)
4. **Chapter 3** — Map Your Thinking (draggable timeline)
5. **Final Reflection Journal** — personalized insights + pattern breakdown + optional journaling

## Tech

- 100% vanilla HTML, CSS, and JavaScript — no frameworks, no build step
- Single file, works fully offline
- Native drag-and-drop with mouse, touch, and keyboard fallback
- Responsive layout, smooth transitions, ambient animation
- Respects `prefers-reduced-motion`
- Fully keyboard accessible (focus states, ARIA roles, live regions)

## Usage

Just open `cognitive-pattern-explorer.html` in any modern browser. No installation, no server, no dependencies.

## Disclaimer

This project is for educational and self-reflection purposes only. It does not diagnose, assess, or replace professional mental health guidance.
