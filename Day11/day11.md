# Day 11 - ATS Resume Optimization with Claude

## Objective

Learn how to optimize a resume for Applicant Tracking Systems (ATS) using Claude AI and improve alignment with target job descriptions.

## Tools Used

* Claude AI
* Resume (PDF/DOCX)
* Job Description
* GitHub
* VS Code

## Process Followed

1. Opened Claude AI.
2. Set effort level to Low.
3. Uploaded resume.
4. Selected a target Job Description.
5. Used ATS Resume Optimizer prompt.
6. Generated ATS Match Score.
7. Reviewed Gap Analysis.
8. Generated ATS-optimized resume.
9. Saved ATS analysis.
10. Saved optimized resume.
11. Captured screenshots.
12. Uploaded all artifacts to GitHub.

## Key Learnings

* ATS systems prioritize keyword matching.
* Job-specific keywords significantly improve resume visibility.
* Quantifiable achievements improve ATS ranking.
* Resume formatting affects ATS parsing accuracy.
* AI tools can rapidly identify missing skills and keywords.

## Deliverables

* ATS Analysis Report
* Optimized Resume
* Screenshots of Results

## Outcome

Successfully analyzed and optimized a resume for ATS compatibility using Claude AI and documented the complete workflow.
