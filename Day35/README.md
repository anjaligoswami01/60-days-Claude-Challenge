# 🧩 Prompt Puzzle — Master AI Prompting Through Play

A single-file, fully offline HTML app that teaches prompt engineering fundamentals through interactive challenges — built as **Day 35** of the **60 Days Claude AI Challenge**.

## 🎯 What It Does

Prompt Puzzle turns prompt engineering concepts into hands-on gameplay across three challenge types:

- **Build the Prompt** — Drag correct prompt components (role, task, tone, constraints) into place while avoiding distractor blocks.
- **Clean the Prompt** — Identify and remove filler/noise from a messy prompt to reveal a clean, effective one.
- **Choose the Best Prompt** — Compare a weak, optimized, and over-engineered prompt side by side and pick the one that reliably produces the desired output.

Each round is scored live on **accuracy, moves, wrong placements, hints used, and an optimization bonus**, culminating in a **Prompt Performance Report** with a score, rank, Prompt DNA breakdown, personalized feedback, and your best optimized prompt from the session.

## 🚀 Getting Started

No installation, server, or internet connection required.

1. Download `prompt-puzzle.html`
2. Open it in any modern browser
3. Pick your round count and start playing

## 🧠 Domain & Difficulty

This build is configured for **Business & Marketing / Beginner** scenarios (cold emails, social captions, product descriptions, HR announcements, etc.), with 8 scenarios randomly sampled per session for replayability.

## 🛠️ Tech Stack

- Pure HTML, CSS, and vanilla JavaScript — zero dependencies, zero build step
- Native drag-and-drop API
- Fully self-contained in one `.html` file

## 💡 Key Takeaway

Across every scenario, the **optimized prompt** (Role + Task + Tone + Constraint) consistently outperformed both the vague **weak prompt** and the bloated **over-engineered prompt** — reinforcing that clarity and restraint matter more than length or complexity in prompt design.
